About this Project

Employee Attendence Using Facial Recognition

Automated employee attendance system implemented using concepts of machine learning. Implementation of the User Interface is done using Web Interface which can be further extended as Web Based Android Application. In Application, Complete details of employee are stored. Each employeeâ€™s facial data is recorded and stored in database. When an employee passes through the camera, his facial data is extracted and compared with the existing data in the database. If match is found, Then Timestamp and snapshot is stored in the database. In this project we are determined to extract a fully automated attendance system using facial recognition and to achieve 90% accuracy in facial recognition to prevent plagiarism and fake identity thus preventing invasion of privacy.
